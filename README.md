# Sbt Inforactive - Office 3D version

## Project setup

```bash
npm install
```

## Run project

```bash
npm run dev
```

## Build project

```bash
npm run build
```

### Deploy project

```bash
npm run deploy
```

### Notes

- This project uses Vite as the build tool.
- The project is set up with React and React Router.
- The project is set up with Tailwind CSS for styling.
